Tag: login

Login

Operation Description
POST /login

Login to the app (cookie is generated)

Tag: logout

Logout of app

Operation Description
GET /logout

Log out of the app (erase all cookies)

Tag: register

Register for app

Operation Description
POST /register

Create a new account (cookie is generated)

Tag: account

User account

Operation Description
POST /account/settings

Edit account info
GET /account/delete

Delete current account you are logged in for

Tag: job

Jobs Available/Accepted

Operation Description
GET /job/show

For clients only, Show all the jobs the client has created
POST /job/create

For clients only, Create a new job (jobs exist already if they exist from the same user and same location)
POST /job/edit

For client only, edit an existing created job (cannot edit location)
POST /job/delete

For client only, delete a posted job at a specific location
GET /job/available

For driver only, Show all unaccepted positions
POST /job/accept

For driver only, Accept a job, and become a driver for that job location
GET /job/accept/show

For driver only, show all accepted jobs
POST /job/cancel

For driver only, cancel being the driver for a particular job
